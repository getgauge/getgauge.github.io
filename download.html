Install Gauge

Mac OS X 

~ $ brew update~ $ brew install gauge

Windows

The windows installer installs gauge to %PROGRAMFILES% by default.

If a custom install location is used, then you need to set GAUGE_ROOT environment variable manually after installation. GAUGE_ROOT should be set as the path to the directory where Gauge is installed.

Linux

Download the zip and execute: 

~ $ unzip gauge-$VERSION-$OS.$ARCH.zip~ $ ./install.sh

Choose the archive file appropriate for your installation. For instance, if you are installing gauge version {{ site.data.version.current }} for 64-bit x86 on Linux, the archive you want is called gauge-{{ site.data.version.current }}-linux.x86_64.tar.gz.

Install Plugins

Post installation you can install the following default Gauge plugins. 

~ $ gauge install html-report

Test your installation

~ $ gauge
