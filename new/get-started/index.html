Get Started
  • Mac
  • Windows
  • Linux

  • There are multiple ways to get started with Gauge on a mac. You can read our detailed documentation for other methods of installation. Here we show you a simple 4 step process to get you running with Gauge on a mac.

    1. Install Gauge

    The table below allows you to download the pkg and follow these instructions to install Gauge offline. You will need to use this method if your setup or organisation does not allow brew. Download the installer below based on your system configuration and follow these detailed instructions to install using the pkg.
    pkg (latest stable version: 0.2.1 )
    32 bit SHA-1: ee81df6fdb335bd7a4ef830c5c770a3b1e42e10d
    64 bit SHA-1: db2d6b7f70152d54e9de429eedc300025e5741a0
    We would recommend using brew for a very simple straightforward installation using this command.
    ~ $ brew update ~ $ brew install gauge

    2. Install Core Plugins

    Gauge has a rich plugin repository. Language runners, IDEs, reports are all plugins. However, since Gauge is a first class command line tool, you only *need* one plugin to get your Gauge tests running. That is your language runner plugin.
    ~ $ gauge --install java
    You can read more about how to install other language runners and plugins here.

    3. Test your installation

    Run this command to check if all plugins are installed and are at the latest version
    ~ $ gauge -v 
    Gauge version: 0.2.0

Plugins
-------
java(0.3.1)

    4. See Gauge working

    Initialise a new Gauge project in the language of your choice. This creates a template with a sample specification for you. The example below shows how you would execute specs in java, but the same applies to all supported languages.
    ~ $ gauge --init java
    Now run the sample specification to see how Gauge works.
    ~ $ gauge specs/hello_world.spec 
    Specifications: 	1 executed, 1 passed, 0 failed, 0 skipped
Scenarios: 	2 executed, 2 passed, 0 failed, 0 skipped
Total time taken: 15.015s
    You're all set. Start testing!

  • There are two ways to install Gauge on windows. We will show you the process using the exe. However using chocolatey is also an option. Read more here on how to install Gauge using chocolatey.

    1. Install Gauge

    Download the exe based on your system configuration and run it. Click your way through till you're asked to select your language. The next section explains what that means.
    exe (latest stable version: 0.2.1 )
    32 bit SHA-1: d67fc2a0c207e16c3920da7511e3acd08b2dc7b3
    64 bit SHA-1: e270c7c92ec1487758cc87f76f7925c72ebc51ac

    Gauge has a rich plugin repository. Language runners, IDEs, reports are all plugins. However, since Gauge is a first class command line tool, you only *need* one plugin to get your Gauge tests running. That is your language runner plugin.

  • Gauge can be installed on any flavour of Linux using the shell script. The following steps will guide you to a quick install on a linux box.

    1. Install Gauge

    Install Gauge with by downloading the zip and execute the following command. Choose the archive file appropriate for your installation. For instance, if you are installing gauge version 0.2.1 for 64-bit x86 on Linux, the archive you want is called gauge-0.2.1-linux.x86_64.tar.gz.
    zip (latest stable version: 0.2.1 )
    32 bit SHA-1: 3c7784f04bfe471d137e13a9b48f75fb8433369c
    64 bit SHA-1: a256b70e237d7ccc136bfbea8968424f04a96bcc
    ~ $ unzip gauge-$VERSION-$OS.$ARCH.zip ~ $ ./install.sh
