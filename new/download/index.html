Installing Gauge

Mac OS X

Option 1 : Using Homebrew

If you have Homebrew installed, you can install Gauge using it.
~ $ brew update
~ $ brew install gauge

Option 2 : Direct Download

Download the Gauge OSx installer and execute: 
~ $ unzip gauge-$VERSION-$OS.$ARCH.zip
~ $ ./install.sh

Windows

Download the Gauge Windows Installer. The windows installer installs gauge to %PROGRAMFILES% by default.

Linux

Download the Gauge Linux installer zip and execute the below 
~ $ unzip gauge-$VERSION-$OS.$ARCH.zip
~ $ ./install.sh
Choose the archive file appropriate for your installation. For instance, if you are installing gauge version 0.1.2 for 64-bit x86 on Linux, the archive you want is called gauge-0.1.2-linux.x86_64.tar.gz.

Install Plugins

Post installation you can install a Gauge plugin using the --install flag 
~ $ gauge --install html-report
Note : IDE plugins need to be installed as per respective IDE's plugin install instructions.

Test your installation

Type gauge in your terminal and hit ENTER. 
~ $ gauge
