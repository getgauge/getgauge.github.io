Contribute

How can I contribute?

Coding is definitely not the only way you can contribute to Gauge. We have listed a few ways of contributing below that would be helpful to the product and community. Take a look below to learn more about how you can become part of the growing Gauge community.

The following are ways to contribute to Gauge.

  • Test Gauge

    You can report an issue you found or you can help us test Gauge and find new issues. Gauge is tested with Gauge. You can also add to our existing automated tests.

  • Discuss with the Community

    We have an active forum(link) where issues, features and ideas are discussed. Start a discussion, or help someone get started with Gauge. Help us build a community so Gauge can help even more people.

  • Blog about Gauge

    Maybe you’ve used Gauge is a different way, or you did an analysis of how a feature works - if you want to write about things like these, then we’d love to add them to the Gauge blog.

  • Enhance our Examples

    We maintain a set of examples that evolve as we add new features and languages. This is to help someone new to Gauge get started easily. If you’ve tried something you think would be useful to the community, please add it.

  • Improve our Documentation

    If you enjoy doing a detailed analysis of how something works whether it's an API or a feature, then please help us improve our documentation. This will really help the Gauge community at large.

  • Code for Gauge

    Gauge has many repositories, all on GitHub. Gauge core is written in Golang. Gauge also has a rich plugin repository, and not everything in there is in Golang. If this is the kind of thing you like, then you can pick up any pending feature/bug, simple or complex. Just make sure you write a test and send us a pull request.

Code Contribution Process

  • Find an issue
  • Assign the issue to yourself
  • Fork the repository, keep it in sync with the master
  • Talk to us about what you want to do
  • We Merge your Request
  • Fill out the CLA Form
  • Submit a Pull Request
  • Make your changes

How do I get help?

We have issues of varying levels across all Gauge repositories, start with something simple and work your way up. All these issues tagged as Help Needed We'd suggest you pick a small bug to start with. These have been tagged as easy picks. If you need any help getting started, then you can talk to us on Gitter or on our discussion forum. We would be happy to help you and answer any questions you might have along the way.

If your contribution is a code contribution, then If you do send us a pull request, we’ll need you read this agreement and sign the Contributor License Agreement.
