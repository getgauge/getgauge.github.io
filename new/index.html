  • Simple Syntax

    Write specifications in Markdown, a plain text, widely used markup. Gauge won't enforce a format: write in the style that works for you.

  • Your Environment
    Your Tools

    You can easily write your own language or IDE plugin. Author plugins in Golang or any other popular programming language. Gauge's plugin-loving architecture helps you get the most out of it.

  • Test code is
    production code

    Test suites need to be as DRY and as malleable as your production code. Gauge's robust refactoring support makes sure your suite evolve as your application does.

  • Parallel execution
    made easy

    Run your tests in parallel with a single command from the IDE or command line. It'll work exactly the same across all languages.
