Get started

1. Download

Download links and install information here

gauge -install
2. Initialize a project
gauge --init [language | java or ruby]

For example

	        
$ gauge --init java
 create  manifest.json
 create  specs
 create  specs/hello_world.spec
 create  env
 create  env/default
 create  env/default/default.properties
 create  src/test/java
 create  libs
 create  src/test/java/StepImplementation.java
 create  env/default/java.properties

Successfully initialized the project. Run specifications with "gauge specs/"

Choose the language the test cases are to written in. This initializes the project. A ruby project structure for example is: 

├── env
│   └── default
│       ├── default.properties
│       └── ruby.properties
├── manifest.json
├── specs
│   └── hello_world.spec
└── step_implementations
    └── step_implementation.rb
