Simple Syntax

Author specifications in markdown, a widely-known markup syntax


Multiple language Support

Write test cases in your favorite programming language


Parallel Execution

Simplest Parallelization strategy across all the languages


Extensible and Hackable

Get more out of Gauge via its plugin architecture. Extension points include: language support, driver support, test execution events, and more.


IDE support

Designed to support IDE integration. Intellij idea integration helps to collaborate better by providing smart suggestions.


Open source

Gauge is released under Gnu General Public License v3 and follows a collaborative development model

Download Gauge
